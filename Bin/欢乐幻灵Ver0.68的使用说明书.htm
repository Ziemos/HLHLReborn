欢乐幻灵Ver0.68的使用说明书

Bien 2003.7.22

功能描述：

  • 创招：自动买药、过劫、删招、保留招
  • 赌币：挂老虎机
  • 瞬移：移动到所选择的地图上
  • 捉宠：捉指定名称的宠，可以选择想要捉宠的级别范围
  • 炼化：编辑UniverseStove.ini文件（详见“UniverseStove.ini说明.htm”）；特点：绝对不会炼错物品，也不怕别人塞物品
  • 九转：一轮做一棵九转

使用方法：

一、首先运行幻灵游侠，进入自己的号

二、运行欢乐幻灵，选择人物，然后点“确定”

三、目前欢乐幻灵实现了6种功能：创招、赌币、瞬移、捉宠、炼化、九转，在“工作模式”处点中相应的单选框， 进入相应的界面

3.1 通用设置

点击“通用设置”按钮，打开“通用设置”面板，可以进行“买药设置”，“战斗设置”和“速度调节”

  1. 买药设置：主要是指在创招时，自动买药的设置，包括到哪个药店买药，以及买什么药；买药前， 如果自己的血没有满，会自动到药店MM那补血
  2. 战斗设置：主要指在过劫、捉宠、九转等功能时，设置人物和宠物的动作；如果工作模式是“捉宠”，则无论一开始的人物选择是什么动作，开始工作后，人物的动作都会被设置成“捉宠”；还可以设置当没有找到想要捉宠的目标时，是“逃跑”，还是“继续战斗”
  3. 速度调节：可以在这里调节工作速度，为了照顾网速慢的朋友

3.2 创招

  1. 点击创招界面的“创招参数”按钮，打开“通用设置”和“创招设置”
  2. “创招设置”处，可以进行“武功参数”，“过劫设置”，“其它设置”
    “武功参数”里面填入你要创招的参数，要与在游戏中实际填入的相同
    “过劫设置”只有在选择“过劫创招”的时候才会起作用；主要可以选择在哪一级过劫
    “其它设置”中，可以选择“从第xx招开始删招”，“降到xx级停止创招”，“过劫创招”；在删招设置中，可以选择是否在删招的时候保留容易招，就是说，不删容易招
  3. 设置好后，选中“开始工作”即可
  4. 可以点击“显示武功”，显示目前人物所有的武功
    注意：“杀伤”那栏，显示的不是实际杀伤，而是在武功属性中，系统显示的杀伤值
  • 在过劫创招的功能中，目前只有地仙1199劫、天仙所有劫、大罗所有劫和天神所有劫的数据

3.3 赌币

先到树城或者水城的赌场，然后点击“位置图示”按钮，参考图示中的位置站好，注意不要站在最边上 的老虎机前面，然后选择“开始工作”即可

3.4 瞬移

选择要到的地方，按“移动”即可进行瞬移，按“停止”，中止瞬移

3.5 捉宠

  1. 输入要捉的宠物名称
  2. 选择想要捉宠的级别范围
  3. 设置好后，来到要捉宠的地方，选择”开始工作“即可
  4. 可以点击“显示宠物”，显示人物身上所带的宠物；“忠”显示的是实际的忠，最高是150
  • 友情提示：工作后，可以在“通用设置”处重新设置人物和宠物的动作，立即生效......

3.6 炼化

  1. 编辑UniverseStove.ini文件（详见“UniverseStove.ini说明.htm”）
  2. 选择”开始工作“即可
  3. 可以“显示工作步骤”
  4. 可以“显示物品信息”，显示炼化时物品在五行炉中的炼化过程
  5. 炼化时，会自动根据UniverseStove.ini中的描述的物品的名称、制作者、属性来判断，不会弄错物品；物品的炼化位置直接在UniverseStove.ini中读取，然后直接写入到内存中，不会弄错
  • 注意1：炼化时，欢乐幻灵会修改幻灵客户端的一些内容，待外挂结束工作后，会恢复
  • 注意2：在炼化的步骤中，欢乐幻灵会屏蔽掉用户对幻灵客户端的操作；待炼化步骤结束后，即可恢复

3.7 九转

  1. 可以显示工作步骤
  2. 注意身上带好徐天麟的英雄令、忠100的觅雾鸠，变成鱼，然后选“工作”即可
  3. 会自动判断身上带什么任务用品，从哪一步开始工作；也就是说，不必非得从第一步开始工作
  • 注意：程序设定只能做一个九转，不过，好像程序中有一个地方有个小问题，我懒得改了......

  1. 将炼化窗口隐藏；在炼化过程中，屏蔽用户对幻灵窗口的操作
  2. 增加了查看物品功能，其中能够看到炼化时五行炉的情况
  3. 增加了重复功能
  4. 修改了买药的逻辑

从Ver0.65到Ver0.67的更新

  1. 增加了炼化的测试功能

从Ver0.6 β1到Ver0.65的更新

  1. 增加了显示人物身上宠物
  2. 取消“只捉一级宠”，让用户指定宠物级别范围
  3. 去掉显示系数，增加显示杀伤（是系统显示的值，不是实际值）
  4. 修正了一些程序上的小错误

从Ver0.5β1到Ver0.6β1的更新

  1. 增加了自动九转任务的功能

从Ver0.4β1到Ver0.5β1的更新

  1. 完善了创招系数的判断
  2. 增加了捉宠功能
